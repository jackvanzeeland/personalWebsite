Basketball Lineup Optimization

Project Overview

I developed a process for a friend who is a basketball coach to help them analyze game data. The tool tracks points scored and allowed for each lineup throughout the game and groups player combinations to identify the most effective lineups, ultimately helping to optimize team performance.

How It Works

The process analyzes player data to determine the most effective lineup combinations on the court. 

                            graph TD;
                                A[Input Game Data] --> B(Track Points Scored/Allowed per Lineup);
                                B --> C(Group Lineups by Size);
                                C --> D(Analyze +/- for each group);
                                D --> E(Identify Most Effective Lineups);
Technologies Used
  • R Programming: For statistical analysis and data manipulation.
  • Problem Solving: To develop the logic for lineup evaluation.
  • Data Manipulation: To process and structure the raw game data.
